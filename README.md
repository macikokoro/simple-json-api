# SIMPLE JSON DATABASE
===============================================

## Set-up
-----------------------------------------------
- Create package.json
- add packages
```
	"name": "node-api",
	"main": "server.js",
	"dependencies": {
		"express": "~4.0.0",
		"body-parser": "~1.0.1"
	}
}
```
*body-parser allows to pull post content from HTTP requests

- $ npm install packages
